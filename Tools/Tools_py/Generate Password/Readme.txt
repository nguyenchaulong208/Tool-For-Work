README - Hướng dẫn sử dụng

1. Mục đích
   Script dùng để sinh mật khẩu ngẫu nhiên có độ dài 8 ký tự,
   bao gồm chữ hoa, số và ký tự đặc biệt.
   Người dùng nhập số lượng mật khẩu cần tạo, kết quả được lưu vào file passwords.txt.

2. Cách chạy
   - Đảm bảo máy đã cài Python (kiểm tra bằng lệnh: python --version hoặc py --version).
   - Đặt các file sau cùng một thư mục:
       + generate_pw.py (script Python)
       + Run.bat (file chạy tự động)
       + README.txt (file hướng dẫn này)
   - Nhấn đúp chuột vào Run.bat để chạy chương trình.

3. Kết quả
   - Chương trình sẽ hỏi số lượng mật khẩu cần sinh.
   - Sau khi nhập, mật khẩu sẽ được tạo và lưu vào file passwords.txt trong cùng thư mục.

4. Cấu trúc thư mục mẫu
   Desktop\
   ├── generate_pw.py
   ├── Run.bat
   └── README.txt
